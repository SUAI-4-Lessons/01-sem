## Установка 

Для установки необходимо склонировать репозиторий и запусить `install.sh`.

```bash
git clone https://github.com/SUAI-4-Lessons/01-sem.git
cd 01-sem
chmod +x install.sh
./install.sh
```

**Будьте внимательны: при запуске установщика файлы перезаписываются**
